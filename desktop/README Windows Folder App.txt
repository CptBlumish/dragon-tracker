Dragon Tracker

This package is the simplest Windows distribution option.

How to use it:

1. Unzip the package.
2. Double-click "Install Dragon Tracker.bat".
3. The installer copies Dragon Tracker into your local app folder, creates a Desktop shortcut, and opens the app.

If you do not want to install it:

- Double-click "Run Dragon Tracker.bat" from inside this folder.

Notes:

- This does not need admin permission.
- Windows may show a warning because this app is not code-signed. Choose "More info" then "Run anyway" if you trust the file.
- Your tracker records are stored on your own computer. Use Backup inside the app before replacing the app with a newer version.
