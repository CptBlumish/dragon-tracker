# Contributing

Dragon Tracker accepts issue reports and feature ideas, but code and release publishing are controlled by the repository owner.

## For Users

- Download the app from the latest GitHub Release.
- Open an issue for bugs or feature requests.
- Do not post backup files, Discord handles, Steam IDs, or screenshots with private account details in public issues.

## For Development

```bash
npm install
npm run check
npm run start
```

Use pull requests for code suggestions. Releases are created from version tags by the owner.
