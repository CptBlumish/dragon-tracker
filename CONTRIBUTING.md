# Contributing

Dragon Tracker accepts bug reports and feature ideas through GitHub Issues.

## Helpful Reports

- Use the bug report template for broken behavior.
- Use the feature request template for new ideas.
- Include your app version from the Backup tab when reporting an issue.
- Describe what you expected and what happened.

## Privacy

Do not post backup files, Discord handles, Steam IDs, or screenshots with private account details in public issues. Code changes and release publishing are controlled by the repository owner.
